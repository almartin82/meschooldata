# Create state-level aggregate from district/school data

Create state-level aggregate from district/school data

## Usage

``` r
create_state_aggregate(district_df, school_df, end_year)
```

## Arguments

- district_df:

  Processed district data frame

- school_df:

  Processed school data frame

- end_year:

  School year end

## Value

Single-row data frame with state totals
